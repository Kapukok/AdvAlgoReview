# AdvAlgoReview

Tarjan R E, Zwick U. *Optimal Resizable Arrays*. SIAM Journal on Computing, 2024, 53(5): 1354-1380. 的阅读报告。

## 分工

熊杺：问题背景、旧方法与空间下界；审查刘畅负责部分

刘畅：(r=3) 数据结构及其 Grow/Shrink 过程、空间与摊还分析；审查丁彦文负责部分

丁彦文：一般 (r) 构造、常数时间访问、空间分析与摊还时间证明；审查杨开云负责部分

杨开云：growth game、时间下界、最优性、局限、现实意义与结论；审查熊杺负责部分
